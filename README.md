## 频段与APN：让你的手机真正跑起来

　　这点太关键了！你有没有遇到过卡插了、钱扣了，但网络死活连不上的情况？我猜多数人是倒在了APN设置上。

　　白俄罗斯主要用LTE Band 3/7/20，国内手机基本都支持。但手动设APN是必须的：
- MTS用户：APN名称填 `mts`，用户名和密码留空；
- A1用户：APN填 `a1by`，其他同样不填。

　　（小贴士：如果上网卡顿，试着把网络模式切换到3G/2G回落到LTE，有时候反而更稳。毕竟他们的4G负载不低。）
